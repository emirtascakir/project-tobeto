# Tobeto Eğitim Platformu Mobil Uygulaması
Bu proje İstanbul Kodluyor eğitiminin bitirme ödevi olarak geliştirilen mobil eğtim platformu uygulamasıdır.

### Kullanılan Teknolojiler
- Dart
- Flutter
- BLoC
- Firebase Auth
- Firebase Firestore
- Firebase Storage
